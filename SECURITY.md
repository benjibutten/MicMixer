# Security policy

## Supported versions

Security fixes are made for the latest release of MicMixer.

## Reporting a vulnerability

Please report vulnerabilities privately through
[GitHub private vulnerability reporting](https://github.com/benjibutten/MicMixer/security/advisories/new).
Include affected versions, reproduction steps, impact, and any suggested fix.

If the private form is unavailable, open a GitHub issue containing only the words
`Security contact requested`. Do not include vulnerability details in a public issue.
The maintainer will arrange a private channel.

Please allow a reasonable amount of time for investigation before public disclosure.
